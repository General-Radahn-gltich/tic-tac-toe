# Tic Tac Toe

A simple Tic Tac Toe game built using React and Vite.

## Features
- 3x3 Tic Tac Toe board
- Reusable React components
- Clean and simple UI

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Run Locally

```bash
npm install
npm run dev
